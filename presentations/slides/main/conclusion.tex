\begin{frame}{Висновки}
    \begin{itemize}
        \item \textbf{Синтез гібридної архітектури:}
            уперше адаптовано SSM для задачі інтерактивної колоризації шляхом
            використання гібридної архітектури U-Net + BiDir-Mamba зі спільними вагами для Mamba.
        \item \textbf{Обчислювальна ефективність:}
            завдяки лінійній складності оптимізованого блоку Mamba досягнуто низького споживання відеопам'яті ($\mathrm{VRAM} = 187.29$ МБ)
            та високої швидкодії ($15.08$ мс), що значно перевершує сучасні важкі аналоги.
        \item \textbf{Висока перцептивна якість:}
            запропонована модель продемонструвала сильну перевагу
            за генеративними критеріями ($\mathrm{FID} = 3.078$, $\mathrm{LPIPS} = 0.143$),
            гарантуючи природність та реалістичність колірної палітри.
        \item \textbf{Бездоганна керованість:}
            рекордно низька похибка ($\mathrm{MSE}_{\mathrm{HINT}} = 6.1 \times 10^{-5}$) підтверджує точне дотримання маркерів користувача
            та відсутність конфліктів під час роботи розробленого програмного комплексу.
    \end{itemize}
\end{frame}
