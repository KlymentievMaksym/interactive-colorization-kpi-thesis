    \textbf{Актуальність дослідження.}
        Стрімкий розвиток методів глибокого навчання відкрив нові можливості для розв'язання складних задач генерації та обробки візуальних даних,
        однією з яких є інтерактивна колоризація монохромних зображень.
        Актуальність розробки нових підходів у цій сфері зумовлена практичним попитом на інтелектуальні інструменти керування кольором для складного фотографічного контенту.
        Класичні методи інтерактивної колоризації на основі математичної оптимізації дозволяють поширювати колір від заданих користувачем підказок,
        проте вони суттєво обмежені відсутністю розуміння високорівневої семантики сцени. 
        Оскільки такі алгоритми спираються переважно на низькорівневі градієнти яскравості, вони вимагають надмірно щільного нанесення маркерів для уникнення «розтікання» кольору за межі об'єктів зі слабко вираженими контурами.
        Натомість впровадження сучасних семантично обізнаних моделей глибокого навчання дозволяє досягти високоякісних результатів за мінімальної кількості підказок.
        Такі інструменти є затребуваними для швидкої генерації багатоваріантних колірних рішень у реальних прикладних задачах, наприклад, в електронній комерції, рекламній фотографії чи дизайні інтер'єрів, де виникає потреба миттєво перефарбовувати окремі об'єкти зі збереженням їхньої природної текстури та тіней.

        Незважаючи на значний прогрес у галузі комп'ютерного зору, існуючі архітектурні рішення мають ряд суттєвих обмежень.
        Традиційні архітектури прямого відображення на базі згорткової нейронної мережі (англ. Convolutional Neural Network, CNN),
        умовної генеративно-змагальної мережі (англ. Conditional Generative Adversarial Network, cGAN) та стандартного візуального трансформера (англ. Vision Transformer, ViT) найчастіше є детермінованими.
        Такі моделі зводять задачу до однозначного математичного відображення, що ускладнює природну генерацію альтернативних кольорових рішень для одного й того ж вхідного зображення.
        
        З іншого боку, стохастичні генеративні архітектури здатні забезпечувати багатоваріантність. 
        Зокрема, генеративно-змагальні мережі (англ. Generative Adversarial Network, GAN) формують різні результати завдяки перетворенню випадкових векторів із латентного простору шуму.
        Своєю чергою, генеративні трансформери, наприклад авторегресивні, забезпечують варіативність за рахунок імовірнісного семплювання токенів із передбаченого розподілу на кожному кроці генерації, 
        а дифузійні моделі (англ. Diffusion Models) досягають цього завдяки ітеративному процесу зняття шуму з різних випадкових початкових станів.
        Однак складність керування та значна ресурсоємність таких систем, що зумовлені важкістю інтеграції точних просторових умов у глобальний латентний простір (у випадку GAN), послідовним характером декодування токенів у поєднанні з квадратичною складністю уваги (у генеративних трансформерах) та потребою у багаторазовому проходженні мережі під час інференсу (у дифузійних моделях),
        роблять їх важкозастосовними для миттєвої інтерактивної взаємодії у реальному часі, де від системи вимагається швидкий відгук на колірні підказки.

        У цьому контексті перспективним напрямком є дослідження новітніх архітектур, зокрема селективних моделей простору станів (англ. Selective State Space Models, SSM).
        На сьогодні рішення на основі SSM для задач інтерактивної багатоваріантної колоризації зображень загального вигляду відсутні,
        а існуючі поодинокі дослідження обмежені вузькоспеціалізованими доменами даних.
        Оскільки чисті архітектури такого класу мають обмежені можливості щодо збереження чітких локальних просторових ознак та текстур зображення,
        виникає обґрунтована необхідність проектування саме гібридних моделей.
        Це зумовлює доцільність поєднання глобального контексту SSM із локальними властивостями згорткових підходів для створення обчислювально ефективних архітектур,
        здатних забезпечити високу якість генерації та миттєвий відгук на дії користувача.

    \textbf{Задача дослідження.}
        Для досягнення поставленої мети необхідно вирішити такі завдання:
        \begin{enumerate}
            % \item провести огляд опублікованих джерел за тематикою дослідження;

            \item проаналізувати сучасний стан досліджень у задачі колоризації зображень та виявити обмеження існуючих методів глибокого навчання, зокрема архітектур на базі CNN, GAN, Transformer та Diffusion;
            \item сформувати набори даних для навчання і тестування, а також розробити алгоритм генерації кольорових підказок для імітації взаємодії з користувачем;
            \item спроектувати та реалізувати власну модель колоризації на базі гібридної архітектури U-Net та B-Mamba, а також інтегрувати базові моделі інших архітектур для порівняння;
            \item здійснити навчання розробленої моделі;
            \item розробити програмне забезпечення із графічним інтерфейсом для виконання інтерактивної багатоваріантної колоризації;
            \item провести комплексне оцінювання її ефективності шляхом порівняльного тестування за об'єктивними метриками і проведення користувацького дослідження.
        \end{enumerate}

        є процес інтерактивної багатоваріантної колоризації зображень.

    \emph{Предметом дослідження}
        є моделі глибокого навчання для колоризації зображень,
        метрики для їх порівняльного аналізу,
        а також методи суб'єктивного оцінювання якості інтерактивної колоризації.

    При розв'язанні поставлених завдань використовувались такі \emph{методи дослідження}:
        методи глибокого навчання та методи комп'ютерного зору,
        методи об'єктивного та суб'єктивного оцінювання якості результатів.

    \textbf{Наукова новизна.}
        Наукова новизна отриманих результатів полягає у наступному:
        \begin{itemize}[label=$\bullet$]
            \item вперше застосовано SSM до задачі інтерактивної багатоваріантної колоризації монохромних зображень загального вигляду,
                що дозволило розширити сферу використання цього класу моделей за межі аналізу вузькоспеціалізованих даних, а саме інфрачервоних та супутникових знімків;
            \item запропоновано нову модифіковану гібридну архітектуру, на базі поєднання U-Net та B-Mamba, для задач інтерактивної колоризації.
                За рахунок інтеграції механізму двонаправленого сканування зі спільними вагами вперше вдалося ефективно об'єднати здатність SSM
                до захоплення глобального контексту з локальною просторовою точністю згорткових мереж, суттєво оптимізувавши споживання відеопам'яті.
        \end{itemize}

    \textbf{Практичне значення.}
        Практичне значення отриманих результатів полягає у створенні швидкого та зручного програмного інструментарію для задач інтерактивної колоризації. Зокрема:
        \begin{itemize}[label=$\bullet$]
            \item розроблена гібридна модель пропонує ефективний компроміс між обчислювальною легкістю та точністю керування. Вона перевершує класичні інтерактивні моделі у точності локалізації підказок і водночас може слугувати швидкою альтернативою важким дифузійним моделям та трансформерам у сценаріях, де пріоритетом є миттєвий відгук у режимі реального часу;
            \item створено програмний застосунок із графічним інтерфейсом, який спрощує користувацьку взаємодію з моделями. Інструментарій дозволяє зручно наносити просторові підказки, генерувати зображення та легко перемикатися між розробленою і базовими архітектурами для прямого візуального порівняння їхніх результатів.
        \end{itemize}
